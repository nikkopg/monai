<objective>
Replace the single-shot `route()`/`ask()` router in `backend/query.py` with a LlamaIndex
`FunctionAgent` + `AgentWorkflow` that plans and chains the existing 9 read tools across
multiple steps in a single turn. This is the foundational vertical slice for the phase:
a user can ask a multi-step financial question and get a synthesized answer with a visible
tool-call trace, while correctness-by-construction (no raw SQL) and honest refusal are preserved.

Establishes the Wave 0 test infrastructure (pytest-asyncio config + async fixtures + the three
new test files) that all subsequent Phase 2 plans build on.

Purpose: Delivers CHAT-01 (multi-step chaining), CHAT-02 (no raw SQL), CHAT-08 (honest refusal) —
the agent reasoning core that Plans 02 and 03 extend with writes and streaming.
Output: Configured async test suite, `test_agent.py` behavior tests, and a rebuilt `query.py`
exposing `agent()`, `agent_stream()`, and an extended `reset_engine()`.
</objective>
