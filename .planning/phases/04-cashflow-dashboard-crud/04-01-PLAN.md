<objective>
Extract every per-operation write branch from `backend/main.py:_execute_proposal_payload` (L197-360) into standalone `apply_*` helper functions in a new `backend/writes.py` module (D-02), then rewire `_execute_proposal_payload` to call them. Purpose: Without a single shared write seam, the direct UI endpoints in Plan 03 would duplicate write logic — a bug fix or Decimal change applied to only one path would silently break the other. This plan makes divergence structurally impossible.

Output: `backend/writes.py` (8 `apply_*` functions) + a `_execute_proposal_payload` rewritten as a thin dispatcher. Zero behavior change on the agent path — proven by the existing Phase 2 test suite staying green.
</objective>
