## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser -> Next.js server | Untrusted client; must not see the API key |
| Next.js server -> FastAPI | Trusted server-to-server call carrying the injected key |
| client -> FastAPI write routes | Untrusted input crossing into state-changing operations |
